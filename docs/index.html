Barb's Hitbox

Barb's Hitbox is a fast, modular, optimized hitbox system for Roblox designed to handle:

  • Spherical hitboxes
  • Box hitboxes
  • Capsule hitboxes
  • Raycast + radius hybrid detection
  • Whitelist / blacklist / team filtering
  • Damage falloff
  • Per-target cooldowns
  • Debug visualization
  • Automatic hitbox controllers

The entire module is open-source and can be freely used in any project.

API Reference

Sphere

BarbsHitbox.Sphere(origin, radius, options)

The fastest and most common hitbox type.

Box

BarbsHitbox.Box(origin, sizeVector3, options)

Capsule

BarbsHitbox.Capsule(origin, radius, height, options)

Raycast

BarbsHitbox.Raycast(origin, direction, length, radius, options)

Options Table

Commonly used properties:

  • Owner — character who owns the hitbox
  • SelfHit — allow hitting yourself
  • HitCooldown — cooldown per character
  • MaxCharacters — max characters per hitbox
  • DebugMode — "All", "HitOnly", "VolumeOnly"
  • Offset — moves the hitbox
  • OnHit — callback with hitData

Usage Examples

Basic Sphere Hitbox

local BarbsHitbox = require(ReplicatedStorage["Barb's Hitbox"]) 

BarbsHitbox.Sphere(workspace.Part, 6, {
  Owner = character,
  Visualize = true,
  OnHit = function(hit)
    hit.Humanoid:TakeDamage(10)
  end
})

Duration Burst Hitbox

local Duration = 0.2
local startTime = os.clock()

RunService.Heartbeat:Connect(function()
  if os.clock() - startTime >= Duration then return end
  BarbsHitbox.Sphere(hrp.CFrame * CFrame.new(0,0,-3), 8, options)
end)

Hitbox Controllers

Create a follow-hitbox

local controller = BarbsHitbox.CreateController(
  "Sphere",
  hrp,
  8,
  nil,
  { Owner = character, Visualize = true }
)

controller:Start(true)
-- controller:Start(false) to stop
