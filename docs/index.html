Overview

This hitbox module is a fully optimized, modular hitbox engine for Roblox games.

  • Spheres / Boxes / Capsules / Rays
  • Whitelist / Blacklist / Team filters
  • Damage callbacks
  • Per-target cooldowns
  • Debug visualization
  • Automatic follow controllers

Hitbox Types

Sphere Hitbox


BarbsHitbox.Sphere(origin, radius, {
  Owner = character,
  Visualize = true,
  OnHit = function(data)
    data.Humanoid:TakeDamage(10)
  end
})

Box Hitbox


BarbsHitbox.Box(origin, Vector3.new(6,4,3), options)

Capsule Hitbox


BarbsHitbox.Capsule(origin, radius, height, options)

Raycast Hitbox


BarbsHitbox.Raycast(origin, direction, length, radius, options)

Options Reference

All hitbox calls share the same options table:


{
  Owner = character,
  Visualize = false,
  DebugMode = "All",
  SelfHit = false,
  MaxCharacters = 10,
  HitCooldown = 0.2,
  Teams = nil,
  Whitelist = nil,
  Blacklist = nil,
  DamageFalloff = function(distance) end,
  OnHit = function(hitData)
    print(hitData)
  end
}

Controllers

Create a Follow-Controller


local controller = BarbsHitbox.CreateController(
  "Sphere",
  hrp,
  8,
  nil,
  {
    Owner = character,
    Visualize = true
  }
)
controller:Start(true)

Examples

Duration Burst Hitbox


local Duration = 0.2
local startTime = os.clock()
RunService.Heartbeat:Connect(function()
  if os.clock() - startTime >= Duration then return end
  BarbsHitbox.Sphere(
    hrp.CFrame * CFrame.new(0, 0, -3),
    8,
    { Owner = character, Visualize = true }
  )
end)
