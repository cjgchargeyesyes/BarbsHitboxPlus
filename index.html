Barb's Hitbox Module

Lightweight, fast, and easy-to-use Roblox hitbox module.

What Is This?

Barb's Hitbox Module is a simple and optimized hitbox system that lets you spawn sphere hitboxes with damage callbacks, cooldowns, and visualization.

  • Fast hit detection
  • Easy API
  • Optional visualizer
  • Supports damage, cooldowns, and duration

API Reference

SmartHitbox.Sphere(position, radius, settings)

-- position: CFrame
-- radius: number
-- settings: table

SmartHitbox.Sphere(pos, 8, {
    Owner = character,
    HitCooldown = 0.3,
    MaxCharacters = 3,
    Visualize = true,
    OnHit = function(hit)
        print("Hit", hit)
    end,
})

Examples

Basic Hitbox

local SmartHitbox = require(ReplicatedStorage["Barb's Hitbox"])

local pos = hrp.CFrame * CFrame.new(0,0,-3)
SmartHitbox.Sphere(pos, 6, {
    Owner = character,
    Visualize = true,
    OnHit = function(hit)
        hit.Humanoid:TakeDamage(10)
    end
})

Duration Example

local duration = 0.25
local start = os.clock()

local conn\conn = RunService.Heartbeat:Connect(function()
    if os.clock() - start >= duration then conn:Disconnect() return end
    SmartHitbox.Sphere(hrp.CFrame, 6, {Owner = char})
end)
